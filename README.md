* Git

* HTML5

# Features

* Five-Day Forecast (displaying the average temperature for the day)
* Switch F to C and C to F
* Responsive Interface
* Refresh
* Error handling 
  - When a country searched is not found
  - Refresh error handling
  - API request failure
* Live preview
